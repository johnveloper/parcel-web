# Parcel Template: Web

## Clone
```
git clone https://github.com/FlutterAddict/parcel-web.git my-simple-webapp
```

## Install dependency
```
npm install
```

## Development
```
npm run dev
```

## Production
```
npm run prod
```

## Push dist/ to GH-Pages
```
git subtree push --prefix dist origin gh-pages
```
